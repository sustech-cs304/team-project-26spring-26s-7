关于 ItsMapPin

ItsMapPin 是一款专为 HarmonyOS NEXT 设计的旅行记录 App。 我们相信旅行是值得被认真记录的——不是堆积一堆照片，而是把每个走过的地方、每段感受都好好留下。

用 AI 让记录更轻松，用签名链接让分享更安全，用本地优先让隐私不让步。 所有功能都为一件事服务：让你能反复回味去过的每个地方。

我们怎么保护你的数据

  • EXIF 元数据完全在端侧抽取，不上传任何照片像素
  • 云端图像识别默认关闭，用户主动开启才送给 AI 服务
  • 分享链接带 HMAC-SHA256 签名 + 过期时间，无法被伪造或延长
  • 所有发布的内容会经过自动审核，违规内容立即下线
  • 账号注销时所有数据可一键清除，不在云端留任何残留
