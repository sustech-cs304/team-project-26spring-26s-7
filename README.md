# TravelPin

![APP海报](references/media/photos/poster.png)

TravelPin 是一个基于 HarmonyOS 的旅行记忆应用，帮助用户将照片和地点绑定到地图上，并按旅行线路进行组织，同时支持生成 AI 旅行文案，方便记录与分享旅途中的精彩瞬间。

本 README 介绍项目的核心功能、安装与配置方法、运行方式，以及几个简单的使用示例，方便新用户快速上手。

## 项目亮点

- 地图记忆标记：在地图上创建带有位置、照片、标题和正文的记忆节点。
- 旅行 Track 管理：按一次旅行组织节点、路线顺序和可见范围。
- 节点集中浏览：既可以从地图查看，也可以从节点列表统一管理。
- AI 文案助手：结合照片元数据、地点和提示词生成中文旅行短文案。
- 华为账号登录与云同步：支持使用 Huawei ID 登录，并将云端数据同步到本地。
- 本地优先存储：当前项目默认接入 RDB 数据服务。
- 分享能力基础设施：项目中已包含分享模块与后端接口配置。
- 多设备无缝同步：记忆节点、路线和草稿通过华为云空间在手机、平板、电脑之间自动同步。
- 跨平台路线分享：支持直接分享至微信，支持直接分享至微信。

## 用户操作指南

### 1. 记忆节点

#### 创建节点

应用提供以下几种方式进入节点创建页面：

| 方式 | 操作 |
|------|------|
| 地图长按选点 | 在首页地图上**长按**目标位置，系统自动反向地理编码获取地名，跳转到创建页并预填坐标与地点名 |
| 地图右下角「+」按钮 | 点击浮动「+」按钮，进入空白创建页，需手动选择地点 |
| 搜索后创建 | 在地图顶部搜索栏选中一个地点后，点击「用该地址新建记忆」按钮 |
| 节点列表页创建 | 切换到「节点」页签，点击顶部「新建节点」按钮 |
| 路线内添加 | 在路线编辑页点击「添加节点」，节点自动归属当前路线 |

**创建页可填写字段：**

- **地点**：必填。预填或点击「重新选点」打开地点选择器
- **所属路线**：选择将节点归入某条路线，也可选择「不关联任何路线」。底部弹出面板支持快速新建路线
- **照片**：最多 9 张，从系统相册选取
- **标题**：必填，最多 20 字
- **正文内容**：最多 1000 字
- **心情**：选择一个表情（😊😢🤩😌🥰😤），可取消
- **标签**：多选（旅行 / 美食 / 风景 / 打卡 / 日常）
- **到访日期**：默认当天，可修改

#### 编辑节点

进入节点详情页后，点击右上角「编辑」按钮。所有字段均可修改：地点、所属路线、照片、标题、正文、心情、标签、到访日期。

#### 删除节点

- **单个删除**：在节点详情页点击右上角「删除」，确认后不可撤销
- **批量删除**：在「节点」页签点击「批量编辑」，勾选目标节点后点击「批量删除」

#### 搜索节点

- **地图页搜索**：顶部搜索栏输入关键词，同时匹配本地节点（标题、地点名、标签）和地图 POI，结果合并展示
- **节点列表页搜索**：顶部搜索栏支持按标题、地点名、正文、所属路线名筛选

#### 将节点加入路线

- 在创建或编辑节点时，通过「选择旅行」选择目标路线
- 在「节点」页签的批量编辑模式下，勾选节点后点击「加入旅行」，批量移动到指定路线或「未分类」

---

### 2. 路线（Track）

#### 创建路线

1. 切换到「旅行」页签，点击「新建 Track」
2. 填写标题，选择可见范围（私密 / 公开，默认私密）
3. 保存后自动进入路线编辑页，可继续添加节点

#### 编辑路线

在「旅行」页签中，每张路线卡片的底部提供两个需要长按封面触发的按钮：

- **查看路线**：进入路线详情页，浏览路线概览、节点列表及地图轨迹
- **编辑**：进入路线编辑页，可修改标题、可见范围，以及对节点进行排序（上移 / 下移）、添加或删除

路线详情页右上角「···」菜单也提供「编辑旅行」和「添加节点」入口。

#### 删除路线

在路线编辑页底部的「危险操作」区域，或在路线详情页的「···」菜单中点击「删除旅行」。**删除路线不会删除其下的节点**——所有节点会被释放为「未分类」状态（不归属于任何路线），节点数据、照片和关联信息完整保留。

#### 搜索路线

在「旅行」页签顶部的搜索栏输入关键词，支持按路线名称、描述、可见范围标签和日期范围筛选。

---

### 3. 分享路线

#### 分享前提

路线必须设置为**公开**才能分享。如果尝试分享一条私密路线，系统会弹窗询问是否自动切换为公开。

#### 分享操作

在路线详情页点击「分享路线」，进入分享页面：

1. 选择链接有效期（5 分钟 / 1 天 / 7 天 / 30 天）
2. 选择分享方式：微信、QQ、海报图片、复制链接
3. 点击发布，系统生成分享链接

#### 让已分享的链接失效

将路线的可见范围从「公开」改回「私密」并保存，系统会自动**撤销该路线下所有已生效的分享链接**。再次生成新链接时，也可选择自动替换旧链接。

### 4. AI 文案功能

#### 使用入口

在节点创建或编辑页面，正文输入区域附近有 AI 文案入口按钮。点击后进入 AI 文案页面。

#### 操作步骤

1. **选择照片**（可选，最多 9 张）：系统从照片中提取 EXIF 信息（拍摄时间、GPS 坐标）、画面构图分析和 OCR 文字识别，用于辅助生成
2. **选择写作风格**：诗意浪漫 / 客观纪实 / 轻松生活 / 极简留白
3. **选择文案长度**：短（1–2 句）/ 中（3–5 句）/ 长（6–10 句）
4. **填写地点名称**（可选，默认从节点预填）
5. **输入额外提示词**（可选，默认从节点正文预填）
6. 点击「生成」

生成后可一键「应用到节点内容并返回」，文案会写入节点的正文区域。

#### 注意事项

- **EXIF 处理方式**：本地照片的 EXIF 数据**不会被剥离**，系统仅从中提取拍摄时间、GPS 坐标用于生成提示词，不会将 GPS 数值直接发送给模型。照片上传到云端时会经过脱敏处理（重新编码为不含 EXIF 的副本，上传后立即删除）
- **网络要求**：生成文案需要联网。离线状态下功能不可用
- **频率限制**：每分钟最多请求 30 次，超出后会收到提示
- **内容审核**：若生成结果涉及敏感内容，系统会提示并替换文案

### 5. 首页地图功能

- **搜索栏**：同时搜索本地节点和地图 POI，选中节点后底部显示预览卡片（封面、标题、地点、标签），选中 POI 后可直接创建节点
- **筛选面板**：点击搜索栏旁的「筛选」按钮，可按**时间范围**（全部 / 近 7 天 / 近 30 天 / 近一年）和**标签**（全部 / 旅行 / 美食 / 风景 / 打卡）组合过滤地图上的标记点
- **标记点交互**：点击单个标记显示预览卡片；点击聚合标记（数字气泡）会展开其中的节点列表并放大地图
- **定位按钮**：右下角定位图标可回到当前 GPS 位置
- **首次进入**：地图自动定位到所有节点的中心点

---

### 6. 回放页面

在路线详情页点击「播放路线」进入回放模式，全屏展示路线动画。

**播放控制**：播放 / 暂停、上一个 / 下一个节点、倍速切换（0.5x / 1x / 2x）、进度条拖拽跳转、背景音乐开关。

**设置面板**（右上角齿轮图标）包含五个分类：

| 分类 | 选项 | 说明 |
|------|------|------|
| **风格** | 极简白 / 暗夜蓝 / 复古胶片 | 整体视觉主题，影响控件、卡片、路线颜色等 30 余项属性 |
| **音乐** | 城市微光 / 晨间鸟鸣 / 夜色爵士 / 热带旅拍 / 静谧电影感 | 背景音乐风格，切换后即时生效 |
| **显示** | 路线留痕 / 玻璃详情层 | 路线留痕：显示完整路线底线并用高亮轨迹点亮已走过的路段；玻璃详情层：展开照片时使用半透明毛玻璃面板 |
| **效果** | 原片 / 胶片 / 暖阳 / 冷调 / 黑白 | 照片叠加滤镜效果 |
| **转场** | 淡入淡出 / 横向滑入 / 轻微缩放 | 节点切换时的过渡动画 |

回放设置会自动保存，分享路线时也会一并上传，使接收方看到与发布者一致的视觉效果。

---

### 7. 离线地图

应用使用华为 Map Kit 渲染地图，可复用花瓣地图（Petal Maps）的离线数据包：

1. 进入「我的」页签，找到「离线地图」设置项
2. 点击后系统引导前往花瓣地图应用
3. 在花瓣地图中进入「我的 → 离线地图 → 地图资源管理 → 地区列表」
4. 下载「全球基础包」以及所需的城市地图

下载完成后，在无网络环境下打开应用即可正常浏览地图和已有标记。

### 8. 华为云服务

#### 自动上传

每次对节点或路线执行创建、编辑、删除操作时，若当前在线，系统会立即将变更**增量同步**到云端。不存在定时轮询，所有同步由操作触发。

#### 手动上传（全量覆盖）

在「我的」页签点击「上传本地」，系统会将本地**所有数据全量推送**到云端并以本地为准覆盖——本地不存在的云端数据会被清除。适用于换设备后首次上传或离线编辑后恢复同步。

#### 手动下载（全量覆盖）

在「我的」页签点击「下载云端」，系统会将云端**所有数据全量拉取**到本地并以云端为准覆盖——云端不存在的本地数据会被清除。适用于在新设备上恢复数据。

#### 离线编辑恢复

如果应用检测到离线期间有过数据变更，重新联网后会在「我的」页签顶部显示提示横幅，提醒用户手动点击「上传本地」将离线改动同步到云端。

#### 相关设置位置

- 云端连接状态、上次同步时间、待同步条目数等信息均显示在「我的」页签的云服务区域
- 「上传本地」和「下载云端」按钮位于同一区域

> **注意**：手动上传和下载均为全量覆盖操作，执行前请确认当前数据是你希望保留的版本。

## 快速上手示例

### 示例 1：创建一个记忆节点

1. 打开应用，进入地图页
2. 点击右侧添加按钮
3. 选择地点
4. 上传或拍摄照片
5. 输入标题和内容（可以选择添加该节点到一条旅行、添加心情、标签和用ai辅助生成文案）

效果：该记忆会作为一个节点显示在地图上。

1. 进入 `旅行` 页签
2. 点击新建 Track
3. 输入 Track 标题
4. 选择可见范围，例如私密或公开
5. 点击保存
6. 保存后继续向 Track 中添加节点

效果：用户可以用这条 Track 来组织整段旅行中的路线、节点顺序和分享范围。

### 示例 3：使用 AI 文案助手

1. 打开 AI 文案助手页面
2. 选择文案长度
3. 填写可选的拍摄地点
4. 输入额外提示词
5. 生成对应文案

效果：系统会根据图片元数据、地点和提示信息生成一段中文旅行文案。

## 已知问题与限制

- 离线地图模式下暂不支持地址解析；如果设备无法访问在线地图或逆地理编码服务，位置可能只能以坐标形式展示。
- 项目域名仍在申请配置中，当前后端使用临时域名与非标准端口
- Replay 回放界面目前采用读条式行进，动画流畅度仍有优化空间。
- 文件传输链路仍以功能可用为优先，大文件或弱网环境下传输效率较低，后续可考虑分片上传、断点续传或压缩策略。
- AI 文案服务受上游 AI 服务能力限制，目前最多支持约 30 个并发请求；超过该并发量时可能触发排队、限流或请求失败。
