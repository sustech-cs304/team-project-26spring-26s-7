# Replay Music Attribution

Record every music asset placed under `references/documents/replay/assets/music/`.

## How To Use

1. Create one section per music file.
2. Use the final repo filename as the section title.
3. Copy the original source URL, not a search results page.
4. Keep the license name or summary text exactly as shown on the source page.
5. Use absolute dates like `2026-05-07`.

## Template

### `<repo_filename>.mp3`
- Source: `<site name>`
- Title: `<track title>`
- Author: `<creator name>`
- Source URL: `<original track page URL>`
- License: `<license name or exact summary label>`
- Commercial Use: `Yes` / `No` / `Check source`
- Attribution Required: `Yes` / `No` / `Check source`
- Modification Allowed: `Yes` / `No` / `Check source`
- Downloaded At: `YYYY-MM-DD`
- Notes: `<optional notes such as mood, why chosen, or restrictions>`

## Example

### `replay_bgm_city_walk.mp3`
- Source: `Pixabay Music`
- Title: `Example Track`
- Author: `Example Author`
- Source URL: `https://pixabay.com/music/...`
- License: `Pixabay Content License`
- Commercial Use: `Yes`
- Attribution Required: `No`
- Modification Allowed: `Yes`
- Downloaded At: `2026-05-07`
- Notes: `Target mood: city walk / lofi / calm daytime`
