# ICP 备案 —— 子域名清单

按你选的"**1 个 audit.itsmappin.top + 路径路由**"方案，备案要填的域名很简洁。

## 一、最终方案的子域名清单

| 域名 | 用途 | 是否必须备案 |
|---|---|---|
| `itsmappin.top` | 主域名（根域名）| **是** |
| `audit.itsmappin.top` | 所有后端 API + H5 viewer 的统一入口 | **是** |

**就这两个。** 其它不需要。

> 为什么不需要 `ai./share./filter./censor.itsmappin.top`：因为我们用 nginx
> **路径路由**，不是子域名路由。所有 4 个后端服务都挂在 `audit.itsmappin.top`
> 的不同 path 下（`/v1/*`、`/api/*`、`/filter/*`、`/censor/*`）。

## 二、华为云 ICP 备案系统里的填法

华为云控制台 → 备案管理 → 新增备案 → 选 ECS `139.159.143.195`：

1. **主体信息**：单位 / 个人（按你们团队真实情况）
2. **网站信息**：
   - 网站名称：`ItsMapPin`（或团队约定）
   - 网站域名：在"域名"输入框里依次填这两条，**它们用同一个备案号**：
     - `itsmappin.top`
     - `audit.itsmappin.top`
   - 网站负责人：填实名（需手持身份证拍照上传）
   - 网站首页 URL：`https://audit.itsmappin.top`
   - 服务内容：选"工具"或"软件 / 信息技术"
3. **服务器信息**：选 `139.159.143.195` 所在 ECS 实例
4. **核验单**：下载 → 法人签字盖章 → 上传

## 三、备案审核期间能不能用

ICP 备案审核期约 **3～20 个工作日**（先管局预审 1-3 天，再通信管理局 7-20 天）。

**审核期间**：
- DNS 已经能解析到 `139.159.143.195`，所以 SSL 终结也能跑（curl 内网测过都对）
- 但**华为云的 80/443 端口会被关闭**，直到 ICP 下来。这是华为云强制策略
- 所以审核期间公网用户**访问不了** `audit.itsmappin.top`

应对：
- 申请 **加急备案**（华为云有付费加急通道，2-5 天）
- 或者审核期间继续用 cloudflared quick tunnel 这条路（不需要 ICP）+ 后端先放校园网
- 一旦 ICP 下来，立即按 [DEPLOY_HUAWEI_CLOUD.md](DEPLOY_HUAWEI_CLOUD.md) 部署 + 切流量

## 四、备案后还要做的

- [ ] 在 H5 viewer 首页 footer 加 ICP 备案号显示，**这是法定要求**：
  ```html
  <a href="https://beian.miit.gov.cn/" target="_blank">
    京ICP备XXXXXXXX号-X
  </a>
  ```
- [ ] 公安备案（"互联网信息服务备案"通过后还需要去
  https://beian.mps.gov.cn/ 做公安备案，30 天内完成）
- [ ] 备案号也要写进应用商店上架时的"网站备案信息"
- [ ] 隐私政策文本里引用备案主体名称

## 五、如果将来想加新子域名（比如 admin.itsmappin.top）

要变更备案，在控制台 → 备案管理 → 找到现有备案 → 申请变更，
把新域名加进去再审一遍（一般 3-7 天）。

所以一开始就规划好，能省事。如果觉得 audit. 一个不够，把要的子域名
**一次性都备案下来**：

| 备选子域名 | 用途（建议）|
|---|---|
| `www.itsmappin.top` | 官网主页（如果做） |
| `admin.itsmappin.top` | 后台管理（如果做） |
| `cdn.itsmappin.top` | 静态资源 CDN（如果做） |

不打算做的就别先备案，备案不用就闲置浪费。
