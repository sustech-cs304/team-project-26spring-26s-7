# 图片处理案例

## 简介

本目录包含图片处理相关的参考项目。

## 源码仓库

**官方仓库**: https://gitee.com/harmonyos-cases/cases

**案例路径**: `CommonAppDevelopment/feature/photo/`

## 快速获取

```bash
# 克隆官方案例仓库
git clone https://gitee.com/harmonyos-cases/cases.git

# 进入本案例目录
cd cases/CommonAppDevelopment/feature/photo/
```

## 说明

本目录原本包含图片预览效果图文件（约 9MB），现为减少仓库体积，仅保留此索引文件。

请参考官方仓库获取完整代码和效果图。

- 图片下载与保存
- 图片编辑与裁剪
