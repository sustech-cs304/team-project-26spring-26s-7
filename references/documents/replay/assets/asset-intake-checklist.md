# Replay Asset Intake Checklist

Use this checklist before adding any asset to the repository.

## Music

- [ ] File format is `mp3`
- [ ] Filename uses stable English snake case, e.g. `replay_bgm_morning.mp3`
- [ ] Duration is suitable for loop playback, ideally `45s-180s`
- [ ] Source page is saved in [music-attribution.md](/D:/Mydata/1University/3Junior/Software_Engineering/project/frontendv1/team-project-26spring-26s-7/references/documents/replay/assets/music-attribution.md)
- [ ] License allows repository use for this course project
- [ ] No unclear third-party copyright claims on the page

## Lottie

- [ ] File format is `json`
- [ ] Source, author, and license are recorded near the file or in a companion note
- [ ] Animation is small enough for mobile use

## Images

- [ ] File format is `png` if transparency is needed
- [ ] Source, author, and license are recorded
- [ ] Asset is not copied from a logo pack or trademarked source
